Github Projects Plus+

This chrome extension adds quality of life features to Github Projects listed below:

  • Organizes all labels, assignees, and milestones in one place for ease.
  • Adds a shortcut that will toggle focus on the card filtering input. You can change the shortcut in the extensions options.
  • Toggle focus on filter - Cmd + Up
  • Clear filters - Ctrl + Down
  • On Mac Ctrl is mapped to Cmd.

This script runs on tabs with the URL that match a Github project URL.

https://github.com/ORG/REPO/projects/NUMBER
